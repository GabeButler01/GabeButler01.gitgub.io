CODING

I have coded for many years. Here are some examples of my projects in several languages.



Generative Art [Python]

Class: Computational Foundations

In this assignment, I created a program that utilized Python's turtle library which drew images of rainbows. The colors of the rainbow, the sky color, and the the clouds in the sky were randomly generated each time. This means that every time the program runs, a unique output is returned.

Blockchain [C++]

Class: Data Structures

This program simulates a simple Blockchain that tracks imaginary exchnages of the cryptocurrency called Buff-CUoins. There are several functions such as: adding a transaction, checking the validity of the blockchain, and mining the pending transactions.

Data Lab [C]

Class: Computer Systems

This assignment involved several puzzles realted to bit-wise operations. Examples of tasks were reversing the bytes of an input, multiplying by 5/8 without risking overflow, and returing a bit-level expression for -f.

Encyclopedia [HTML]

Class: Web

To practice structuing an HTML document, this exercise involved correctly using elements and tags. It also had to pass the W3C Markup validator, impliment absolute and relative links along with other miscellaneous functionalities.

CSS Layouts [CSS]

Class: Web

To get familiar with CSS layouts, this activity involved creating a grid, a navigation bar, and wrapping text around an image. This practice set up the foundation for this website.

Bibliography Organizer [Java]

Class: AP/IB Computer Science A

For my IB Internal Assessment, I decided to create a bibliography organizer. It takes an input of either an annotated or unannoted bibliography then alphabetizes it and adds hanging indents.
