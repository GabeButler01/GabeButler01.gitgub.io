Generative Art

Class: Computational Foundations

In this assignment, I created a program that utilized Python's turtle library which drew images of rainbows. The colors of the rainbow, the sky color, and the the clouds in the sky were randomly generated each time. This means that each time the program is run, a unique output is returned.
